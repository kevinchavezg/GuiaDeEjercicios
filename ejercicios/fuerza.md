# Fuerza

## Descripción
El entrenamiento de fuerza implica el uso de resistencia (pesas, bandas o el propio peso corporal) para aumentar la masa muscular, la potencia y la resistencia física.

## Beneficios
- Aumenta la densidad ósea y muscular.
- Acelera el metabolismo basal.
- Mejora la postura y protege las articulaciones.
