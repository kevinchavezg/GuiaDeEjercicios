# Funcional

## Descripción
El entrenamiento funcional se basa en movimientos que imitan las actividades cotidianas, trabajando múltiples cadenas musculares de forma simultánea.

## Beneficios
- Mejora el equilibrio, la coordinación y la agilidad.
- Fortalece el "core" (zona central del cuerpo) de manera integral.
- Optimiza el rendimiento para las tareas del día a día.
